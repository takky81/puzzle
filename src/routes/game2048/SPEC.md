# 2048 仕様書

## 概要

4x4のグリッド上で数字タイルをスライドさせ、同じ数字同士を合体させて2048タイルを目指すパズルゲーム。

## ルール

- 4x4のグリッドでプレイ
- 上下左右いずれかの方向に全タイルをスライド
- 同じ数字のタイル同士がぶつかると合体（2+2=4, 4+4=8, ...）
- 1回のスライドで各タイルが合体できるのは1回まで
- スライド後、空きマスにランダムで新しいタイルが出現（値と位置は難易度による）
- 2048タイルができたらクリア（続行も可能）
- タイルが動かせなくなったらゲームオーバー

## 難易度

- **Easy**: 隣接セルに同じ値のタイルがある空きマスを優先配置。新タイルは100%「2」
- **Random**: 空きマスからランダム配置。90%「2」/ 10%「4」
- **Hard**: 以下の優先順位で配置位置を決定。新タイルは60%「2」/ 40%「4」
  1. 新タイル配置後に動かせる方向が最も少なくなる位置
  2. 追加したタイルを次のターンでマージできない位置と値
  3. 隣接するタイルの合計値が大きい位置（空きマスがある場合はその先の隣接タイルを参照）

## UI仕様

- **グリッド**: 4x4、角丸の各セルに数字タイル
- **タイルの色**: 数字に応じて色が変化（2=薄い, 1024+=濃い）
- **スコア表示**: 現在のスコア（合体時の数値を加算）
- **ベストスコア**: ローカルストレージに保存（難易度共通で1つ）
- **アニメーション**: タイルのスライドと合体時のアニメーション
- **メッセージ**: クリア時「2048達成!」、ゲームオーバー時「Game Over」
- **New Gameボタン**: ゲームをリセット
- **難易度選択**: Easy / Random / Hard の切替（ゲーム中でも変更可能、ボードはそのまま継続）
- **undo/redoボタン**: 1手戻す・やり直す（無制限、undo時はスコアも戻る）

## 操作方法

- **スマホ**: スワイプで上下左右にスライド
- **PC**: 矢印キーまたはWASDキーで上下左右にスライド

## レイアウト

- グリッドは画面幅に合わせて正方形を維持（max-width: 400px程度）
- グリッドの上にスコアとベストスコア
- グリッドの下にNew Gameボタン
